GUESS THE NUMBER GAME

Guess the number is a simple game where you have to guess a number between 1 and 100. You have 5 attempts to guess the number correctly. If you guess the number correctly, you win the game. If you don't guess the number correctly after 10 attempts, you lose the game.

The game is designed to help you improve your guessing skills. It is a fun and interactive game that you can play with your friends and family. The game is suitable for all ages and can be played on any device.

To play the game, simply enter a number between 1 and 100 in the input field and click the submit button. The game will tell you if your guess is too high, too low, or correct. You can also restart the game at any time by clicking the restart button.

Have fun playing the game and see if you can guess the number correctly in 5 attempts or less!
